## Story

**As a** Ana Paula (admin do sistema),  
**I want** executar um script CLI que gere automaticamente descrições de marketing para todos os 435 produtos usando Claude Vision,  
**so that** a Gabi e o time de e-commerce tenham textos persuasivos e prontos para uso em redes sociais e marketplaces, sem precisar escrever manualmente produto a produto.

---

## Acceptance Criteria

1. Script `scripts/enrich-marketing.ts` executável via `npx ts-node scripts/enrich-marketing.ts`
2. Processa produtos em batches de 10 (respeitar rate limit Anthropic)
3. Flag `--dry-run`: exibe output gerado sem gravar no banco
4. Flag `--force`: re-processa mesmo produtos que já têm `descricao_marketing`
5. Flag `--sku {SKU}`: processa apenas um produto específico
6. Flag `--estimate`: exibe custo estimado total antes de executar (sem processar)
7. Log de progresso por produto: `[N/435] SKU: LBG100IPANEMA → OK (tempo: 1.2s) | ERRO: {msg}`
8. Log final: total processados, total com erro, custo estimado Anthropic em USD
9. Descrições geradas: 1–3 frases, tom elegante e confiante, foco no diferencial visual do produto
10. Não sobrescreve `descricao_tecnica` — atualiza APENAS `descricao_marketing`
11. Produtos sem `image_url` são pulados com log: `[SKIP] SKU sem imagem`
12. Rate limit respeitado: máx 10 req/min para API Anthropic (delay automático entre batches)
13. Falha em produto individual não interrompe o batch — continua para o próximo
14. TypeScript compilando sem erros (`npm run typecheck`)
15. ESLint sem erros (`npm run lint`)

## Dev Notes

### Padrão de chamada Anthropic (extraído de `src/app/api/upload/route.ts`)

```typescript
import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

// Chamada com imagem URL (não base64 — performance melhor)
const response = await client.messages.create({
  model: 'claude-sonnet-4-6',
  max_tokens: 150,
  messages: [{
    role: 'user',
    content: [
      { type: 'image', source: { type: 'url', url: imageUrl } },
      { type: 'text', text: promptMarketing }
    ]
  }]
});

const texto = response.content[0].type === 'text' ? response.content[0].text : '';
```

**Atenção:** Verificar `response.content[0]` com guard `Array.isArray` antes de acessar (bug H2 corrigido em v4.4).

### Supabase Server (script Node.js, não Next.js API route)

No script standalone, importar `supabaseServer` de `@/lib/supabase` pode não funcionar com alias. Usar caminho relativo ou instanciar diretamente:

```typescript
import { createClient } from '@supabase/supabase-js';
import * as dotenv from 'dotenv';
dotenv.config({ path: '.env.local' });

const sb = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL!,
  process.env.SUPABASE_SERVICE_ROLE_KEY!
);
```

### Rate Limit Anthropic (FREE tier)

- Tier gratuito: ~60 req/min para Claude Sonnet
- Batch de 10 com delay de 6s entre batches = seguro no free tier
- `Promise.allSettled()` para não deixar falha individual quebrar o batch

### Prompt de Marketing (template base)

```
Você é um copywriter especializado em produtos hidráulicos premium.
Analise esta imagem de produto da La Bella Griffe e escreva 1-3 frases de marketing em português.
Tom: elegante, confiante, foco no design e qualidade visual.
Máximo 200 caracteres no total.
Não inclua preço, SKU ou código.
```
