## Story

**As a** usuária do catálogo LBG,  
**I want** ver o rótulo do ângulo abaixo de cada thumbnail na galeria do modal,  
**so that** eu saiba qual thumbnail clicar para ver o ângulo frontal, lateral ou perspectiva sem precisar passar o mouse por cima.

---

## Contexto

Atualmente os thumbnails no `ProductModal` (`src/app/catalogo/page.tsx`, linhas 268-295) mostram o ângulo apenas como atributo `title` (tooltip de hover). Em dispositivos mobile ou touch, esse tooltip não é acessível. Além disso, o usuário tem que "adivinhar" qual foto é frontal vs lateral.

A solução é adicionar um mini-label de texto abaixo de cada thumbnail com o nome do ângulo (ex: "frontal", "lateral"). O label deve ser truncado para não ultrapassar os 60px de largura do thumbnail.

---

## Notas de Implementação

Mudança cirúrgica dentro do `map` de thumbnails. Estrutura após a mudança:

```tsx
<button key={foto.id} onClick={() => setIdx(i)} style={{ ...existingStyles, height: 'auto' }}>
  {/* imagem existente — mantida */}
  <div style={{ width: 60, height: 60, overflow: 'hidden', borderRadius: 4, border: i === idx ? '2px solid #b45309' : '2px solid transparent' }}>
    {imgUrl(foto.image_url, 120)
      ? <img ... />
      : <span ...>📷</span>
    }
    {foto.precisa_revisao && <span style={{ position:'absolute', top:2, right:2, fontSize:'0.6rem' }}>⚠️</span>}
  </div>
  {/* label novo */}
  <div style={{ fontSize:'0.55rem', textAlign:'center', color:'#78716c', overflow:'hidden',
                textOverflow:'ellipsis', whiteSpace:'nowrap', maxWidth:60, marginTop:2, textTransform:'capitalize' }}>
    {foto.angulo ?? '—'}
  </div>
</button>
```
