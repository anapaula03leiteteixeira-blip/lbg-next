## Contexto

O script `scripts/generate-copies.ts` já existe (EPIC-1 Story 1.4) com toda a infraestrutura de batch processing, rate limiting, UPSERT no Supabase e flags CLI (`--dry-run`, `--force`, `--estimate`, `--sku`, `--plataforma`). O que falta é inteligência nos prompts: os prompts atuais em `buildPrompt()` são genéricos e não incorporam as regras SEO de cada plataforma.

Esta story atualiza `buildPrompt()` e `contextoBase()` com prompts refinados baseados em pesquisa de algoritmos de cada plataforma (Atlas/2026-06-07).

**Arquivo a modificar:** `scripts/generate-copies.ts` (projeto lbg-next)

---

## User Story

Como operador do sistema, quero executar `npx ts-node --project tsconfig.scripts.json scripts/generate-copies.ts --dry-run --sku LBG100IPANEMA` e receber copies SEO que respeitam os limites e padrões de cada plataforma (keywords no início do título Shopee, bullet points estruturados Amazon, ficha técnica para ML, etc.), para que os listings gerados maximizem visibilidade e conversão.

## Notas Técnicas

**COPY_LIMITS após mudança:**
```typescript
const COPY_LIMITS = {
  amazon:          { titulo: 200, bullets: { count: 5, chars: 255 }, descricao: 2000 },
  mercado_livre:   { titulo: 60,                                      descricao: 4000 },
  shopee:          { titulo: 58,                                       descricao: 3000 },
  leroy_merlin:    { titulo: 120, bullets: { count: 10, chars: 200 }, descricao: 280  },
  madeira_madeira: { titulo: 120,                                      descricao: 2000 },
};
```

**Query expandida — incluir produtos sem descricao_marketing:**
```typescript
let query = sb
  .from('produtos')
  .select('sku, nome_produto, categoria, subcategoria, cor_dominante, material_aparente, tags, descricao_marketing, descricao_tecnica')
  .not('nome_produto', 'is', null);  // apenas exclui registros sem nome
// Remover .not("descricao_marketing", "is", null) e .neq("descricao_marketing", "")
```

**CopyRaw — adicionar campos novos:**
```typescript
interface CopyRaw {
  titulo?: string;

  descricao?: string;
  palavras_chave?: string[];
  ficha_tecnica_hints?: string[];   // ML + Leroy
  backend_keywords?: string;         // Amazon
}
```
